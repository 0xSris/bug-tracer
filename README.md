# bug-tracer

A lightweight **session recorder & replay tool** for web pages.  
Records clicks, inputs, scrolls, and console events, then replays them visually with a neon-green cursor.   

---

## Features

- Record clicks, inputs, and scroll events
- Export sessions as JSON
- Replay events with visual cursor
- Timeline scrubber & playback speed control
- Dark mode UI with neon-green accents
